# cisco.nxos.switchportConfig

**Description**

Cisco NX-OS Switchport configuration

**Fields**

| ID              | TYPE   | DESCRIPTION                                |
| --------------- | ------ | ------------------------------------------ |
| enabled         | bool   | Whether the switchport is enabled.         |
| monitorEnabled  | bool   | Whether the switchport monitor is enabled. |
| operationalMode | string | The operational mode of the switchport.    |
