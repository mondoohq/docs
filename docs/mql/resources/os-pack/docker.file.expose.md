# docker.file.expose

**Description**

Dockerfile EXPOSE instruction

**Fields**

| ID       | TYPE   | DESCRIPTION                                                    |
| -------- | ------ | -------------------------------------------------------------- |
| port     | int    | Port that is exposed                                           |
| protocol | string | Protocol that is exposed (evaluates to `tcp` if not specified) |
