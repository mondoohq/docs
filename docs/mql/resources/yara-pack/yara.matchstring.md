# yara.matchString

**Description**

A YARA match string represents a string declared and matched in a rule

**Fields**

| ID         | TYPE   | DESCRIPTION       |
| ---------- | ------ | ----------------- |
| name       | string | string identifier |
| lineNumber | int    | line number       |
| offset     | int    | offset            |
| length     | int    | length            |
| data       | string | data              |
