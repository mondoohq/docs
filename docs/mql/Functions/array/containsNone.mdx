# `containsNone`

Check that an array contains none of the specified values. Returns `true` if no matches are found.

---

## Examples

Ensure no insecure packages are installed:

```coffee
packages.map(name).containsNone(["telnet","rsh"])
```

Example output:

```
packages.map(name).containsNone(["telnet","rsh"])
[failed] [].containsNone()
  expected: == []
  actual:   [
    0: "telnet"
  ]
```
