The Mondoo AWS integration lets you continuously monitor your AWS account, EC2 instances, EKS clusters, EBS volumes, and more. Choose between a Mondoo-hosted integration and a serverless integration.

| ****                                     | **Mondoo-hosted**        | **Serverless**                                     |

| **Complexity**                           | Easy                     | Requires installation in your environment          |
| **Cost**                                 | No additional cost       | Small AWS cost                                     |
| **Security**                             | High                     | Highest                                            |
