# macos.hardware

**Description**

macOS hardware overview

**Fields**

| ID                   | TYPE   | DESCRIPTION                                                 |
| -------------------- | ------ | ----------------------------------------------------------- |
| activationLockStatus | string | Activation Lock security feature                            |
| bootRomVersion       | string | Boot ROM (firmware) version identifier                      |
| chipType             | string | Processor chip type (e.g., "Apple M1", "Apple M2", "Intel") |
| machineModel         | string | Specific model identifier (e.g., "MacBookPro18,3")          |
| machineName          | string | User-friendly hardware name (e.g., "MacBook Pro")           |
| modelNumber          | string | Apple's model number for the device (e.g., "A2442")         |
| numberProcessors     | string | Total number of processor cores                             |
| osLoaderVersion      | string | Version of the OS bootloader                                |
| physicalMemory       | string | Total RAM installed (e.g., "16 GB")                         |
| platformUUID         | string | Universally unique identifier for the hardware platform     |
| provisioningUDID     | string | Unique Device Identifier for MDM provisioning               |
| serialNumber         | string | Apple's unique serial number for the device                 |
