# Yara MQL Resource Pack Reference

The YARA resource pack lets you use MQL to query and assess the security of your infrastructure by using YARA rules.

Resources included in this pack:

| ID                                      | DESCRIPTION                                                                      |
| --------------------------------------- | -------------------------------------------------------------------------------- |
| [parse.yara](parse.yara.md)             | Parse a YARA ruleset files                                                       |
| [yara.matchRule](yara.matchrule.md)     | A YARA match rule represents a rule successfully matched against a block of data |
| [yara.matchString](yara.matchstring.md) | A YARA match string represents a string declared and matched in a rule           |
| [yara.rule](yara.rule.md)               | YARA rule                                                                        |
| [yara.ruleset](yara.ruleset.md)         | A YARA ruleset represents the content of yar file                                |
| [yara.rulesets](yara.rulesets.md)       | YARA Rule Sets                                                                   |
| [yara.scan](yara.scan.md)               | Scan a file                                                                      |
