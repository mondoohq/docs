# shodan.nsrecord

**Description**

Shodan Search Engine DNS NS record

**Fields**

| ID        | TYPE   | DESCRIPTION         |
| --------- | ------ | ------------------- |
| domain    | string | DNS domain          |
| subdomain | string | DNS subdomain       |
| type      | string | DNS record types    |
| value     | string | DNS record value    |
| lastSeen  | time   | Last seen by Shodan |
