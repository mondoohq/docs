# fortios.system.firmware.versionInfo

**Description**

FortiOS System Firmware Version Information

**Fields**

| ID          | TYPE   | DESCRIPTION                           |
| ----------- | ------ | ------------------------------------- |
| id          | string | Image version identifier or "current" |
| name        | string | Name of the firmware image            |
| version     | string | Semantic version of the firmware      |
| major       | int    | Semantic version's major component    |
| minor       | int    | Semantic version's minor component    |
| patch       | int    | Semantic version's patch component    |
| build       | int    | Build number of the firmware          |
| branchPoint | int    | Branch point                          |
| releaseType | string | Release type                          |
| maturity    | string | Maturity level                        |
| notes       | string | Notes                                 |
| source      | string | Source                                |
| platformId  | string | Platform ID                           |
