# cisco.nxos.sshConfig

**Fields**

| ID        | TYPE   | DESCRIPTION                                                     |
| --------- | ------ | --------------------------------------------------------------- |
| algorithm | string | The SSH key algorithm used (e.g., rsa, ecdsa or dsa).           |
| keySize   | int    | Optional. The SSH key size in bits. Null when algorithm is DSA. |
