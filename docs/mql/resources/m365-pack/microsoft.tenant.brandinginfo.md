# microsoft.tenant.brandingInfo

| domainName             | string | Domain name                                       |
| keepMeSignedInDisabled | bool   | if "Keep me signed in" is disabled for the tenant |
| backgroundColor        | string | Tenant branding background color                  |
| bannerLogo             | string | Tenant branding logo URL                          |
| useTransparentLightbox | bool   | Transparent light box                             |
